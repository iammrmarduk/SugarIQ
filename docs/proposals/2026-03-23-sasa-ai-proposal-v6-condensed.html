Programme Sessions

Six Sessions, One Direction

S1 — Week 1 Foundations Audit, Strategic Framing & Prompt Engineering
  • Map SASA data landscape: R7106, LIMS, Autolab, mill reports
  • Configure tooling and shared Claude Pro account
  • Prompt engineering fundamentals: RCTF anatomy, XML tags, 200K context window
  • Establish shared prompt library and consulting log
  • Deliverable: systems map + seed prompt library (10–15 entries)
S2 — Week 3 Excel, Reporting & AI-Generated Workflows
  • Rebuild SASA Excel report using Claude as co-pilot
  • Reusable Excel prompt library and exception-flagging template
  • Copilot Excel + PowerPoint integration
  • Repeatable report narrative workflow: data → prompt → draft → director sign-off
  • One complete AI-assisted report draft produced in-session
S3 — Week 5 Large Datasets, RAG & Context Management
  • RAG: query large SASA datasets without uploading everything
  • Context window management: paste vs. chunk vs. RAG
  • Chunking and summarisation chains for 10-year multi-centre data
  • SASA data-preparation checklist (LIMS, Autolab, mill formats)
  • Deliverable: RAG demo + context reference card + checklist
S4 — Week 7 Python & Automation Pipelines
  • AI-generated Python pipeline: LIMS export → transformation → flagged output
  • Node.js for API and SharePoint connectors
  • Version control basics for AI-generated code
  • Automation candidate list per team area (input to Session 6 build day)
  • Deliverable: working LIMS pipeline stub + candidate list
S5 — Local AI + Agents Local AI Infrastructure & Agentic Workflows
  • Install local AI model on SASA server (Ollama + Open WebUI) — no software on individual machines
  • Local RAG: query internal SASA documents with no internet
  • SASA data governance policy: cloud vs. on-premise classification
  • Intro to agents: AI that does tasks, not just answers questions
  • Build a SASA agent pipeline: data → exception check → alert → SharePoint output (n8n)
S6 — Build Day + Close Build Day, Governance & Programme Roadmap
  • Teams build one working tool from their automation candidate lists
  • Showcase: each group presents their output (demo, not slides)
  • SASA AI governance framework and operating model
  • 12-month integration roadmap + individual 90-day action plans
  • Goal: every group ships something usable from Monday

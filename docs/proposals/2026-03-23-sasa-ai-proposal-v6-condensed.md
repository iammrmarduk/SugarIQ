## Programme at a Glance

### Session 1 — Foundations Audit, Strategic Framing & Prompt Engineering
- Map SASA data landscape (R7106, LIMS, Autolab, mill reports)
- Configure tooling and shared Claude account
- Prompt engineering fundamentals: RCTF anatomy, XML tags, 200K context window, iteration workflow
- Establish shared prompt library and consulting log

### Session 2 — Excel, Reporting & AI-Generated Workflows
- Rebuild a SASA Excel report using Claude as co-pilot
- Build reusable Excel prompt library (summarise, flag, extract, compare)
- Design exception-flagging template for one reporting area
- Copilot Excel and PowerPoint integration
- Build repeatable report narrative workflow: data → prompt → draft → director sign-off
- Produce one complete AI-assisted report draft, reviewed in-session

### Session 3 — Large Datasets, RAG & Context Management
- Retrieval Augmented Generation: query large SASA datasets without uploading everything
- Context window management: when to paste, chunk, or use RAG
- Chunking and summarisation chains for 10-year multi-centre data
- SASA data-preparation checklist for LIMS, Autolab, and mill formats

### Session 4 — Python & Automation Pipelines
- AI-generated Python pipeline: LIMS export → transformation → flagged output
- Node.js for API and SharePoint connectors
- Version control basics for AI-generated code
- Automation candidate list per team area (input to Session 6 build day)

### Session 5 — Local AI Infrastructure & Agentic Workflows
- Install local AI model on SASA server (Ollama + Open WebUI)
- Browser access for all team members — no software on individual machines
- Local RAG: query internal SASA documents with no internet connection
- SASA data governance policy: cloud vs. on-premise classification
- Introduction to agents: AI that does tasks, not just answers questions
- Build a SASA agent pipeline: data → exception check → alert → SharePoint output (n8n)

### Session 6 — Build Day, Governance & Programme Roadmap
- Teams build one working tool from their automation candidate lists (morning)
- Showcase builds; each group presents their output
- SASA AI governance framework: acceptable use, data classification, prompt library ownership
- 12-month integration roadmap + individual 90-day action plans
